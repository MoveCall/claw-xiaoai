# clawra-xiaoai

Clawra Xiaoai — an OpenClaw companion skill that adds a character-driven persona, selfie generation, visual consistency, and natural companion-style interactions.

## Included

- `clawra-xiaoai/` — source skill folder
- `dist/clawra-xiaoai.skill` — packaged distributable skill

## Features

- Clawra persona prompt and companion behavior
- Selfie trigger mapping (direct / mirror)
- Stable visual identity anchors
- Caption style guidance
- ModelScope-based selfie generation script
- Starter config generator

## Main Files

- `clawra-xiaoai/SKILL.md`
- `clawra-xiaoai/references/clawra-prompt.md`
- `clawra-xiaoai/references/visual-identity.md`
- `clawra-xiaoai/references/caption-style.md`
- `clawra-xiaoai/scripts/build_clawra_prompt.py`
- `clawra-xiaoai/scripts/generate_selfie.py`
- `clawra-xiaoai/scripts/generate_clawra_config.py`
